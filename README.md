# Soportes de Extintor para Pared

## Base de Extintor para Gabinete
Un soporte metálico diseñado para colocar un extintor dentro de un gabinete de seguridad.

**Características:**
- Material resistente.
- Ajuste firme dentro del gabinete.
- Fácil instalación.

[Leer más](#)

---

## Chapa Push para Gabinete
Un sistema de cierre con mecanismo de empuje para asegurar el gabinete del extintor.

**Características:**
- Bloqueo de seguridad.
- Previene aperturas accidentales.
- Diseño compacto y duradero.

[Leer más](#)

---

## Gabinete Metálico de Extintor
Gabinete de seguridad fabricado en metal disponible en diferentes tamaños.

**Características:**
- Protección robusta contra golpes y humedad.
- Variedad de tamaños según necesidad.
- Fácil acceso al extintor en emergencias.

[Leer más](#)

---

## Gabinete de Plástico Chico
Gabinete protector de plástico en tamaño compacto.

**Características:**
- Resistente al desgaste y al clima.
- Ligero y fácil de manejar.
- Ideal para espacios reducidos.
